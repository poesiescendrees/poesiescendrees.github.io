la base

Voici la page de visualisation du package CSS. Basiquement, toute la fiche de style qui est utilisée sur cette page est disponible dans le code source. Des classes et balises HTML sont prédéfinies afin d'assurer l'esthétique des blocs qu'elles définissent ainsi que leur placement. Cela permet que vous puissiez les utiliser dans des situations les plus larges possibles. Si vous avez certaines notions de base en HTML et en CSS, vous ne devriez pas avoir de grosse difficulté à prendre le code en main.

Le code se veut être une base à votre site, qu'il s'agisse d'un forum, d'une page Tumblr ou même autre chose. Pour cela, le fichier CSS prédéfinit un certain nombre de couleurs et de variables que vous pouvez modifier à votre guise. C'est plus logique si leur utilisation se fait sur l'ensemble du site, et non seulement pour les balises HTML mises en service, mais cela reste à votre discrétion.

Un panel de modification des couleurs est disponible si vous cliquez sur l'icône dans le coin supérieur gauche de cette fenêtre; cela vous permettra de visualiser ce que le pack de CSS peut donner avec un autre design. Le code de libre service comporte cependant les couleurs de base, telles qu'affichées par défaut ici.
