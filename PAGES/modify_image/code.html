Codes HTML d'images

Vous pouvez utiliser cette page pour créer un code contenant les images dont les URLs sont copiés ci-dessous, et spécifier si voulu une taille fixe pour toutes ces images.

Vous pouvez y placer uniquement les URLs (séparés par un espace ou un retour à la ligne) ou y mettre un code complet (tel que ceux donnés en hébergeant vos images sur ImgBox par exemple, en HTML ou BBcode). Seuls les URLs d'images png, gif et jp(e)g seront détectés.
